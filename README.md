## Core Language Fundamentals

### Variables & Types
- Type system (sound null safety)
- Built-in types: `int`, `double`, `String`, `bool`, `List`, `Set`, `Map`
- `var`, `final`, `const`, `late`
- Type inference
- Dynamic vs Object types

### Functions
- Function declarations & expressions
- Named vs positional parameters
- Optional parameters (positional & named)
- Default parameter values
- First-class functions & higher-order functions
- Anonymous functions (closures)
- Lexical scope & closures
- `Function` type

### Operators & Expressions
- Cascade notation (`..`)
- Spread operator (`...`, `...?`)
- Collection `if` & `for`
- Null-aware operators (`?.`, `??`, `??=`, `...?`)
- Type test operators (`is`, `is!`, `as`)
- Conditional expressions

### Control Flow
- `if`, `else if`, `else`
- `switch` & `case` (enhanced)
- `for` loops (standard, for-in, forEach)
- `while` & `do-while`
- `break`, `continue`, labels
- Pattern matching (Dart 3.0+)
